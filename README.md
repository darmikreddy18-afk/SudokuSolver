# Sudoku Solver (Java)

A Sudoku solving application implemented in **Java** using **Swing GUI** and **backtracking algorithm**.  


## Features

- Displays a **predefined Sudoku puzzle** on launch.  
- **Solve** the puzzle automatically with backtracking.  
- **Reset** to original puzzle.  
- Detects **invalid or unsolvable puzzles**.  


## Skills Demonstrated

- **Data Structures & Algorithms**: Backtracking, 2D arrays.  
- **GUI Development**: Java Swing.  
- **Problem Solving**: Recursive Sudoku solver.
